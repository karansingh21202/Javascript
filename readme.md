

# To-Do List Web App

## 1. Overview

Welcome to the To-Do List Web App repository! This project offers a simple and user-friendly to-do list application built using HTML, CSS, and JavaScript. It empowers users to effectively manage their tasks by adding, checking off, and removing items from their to-do list.

## 2. Features

- Task Addition: Effortlessly add tasks by typing them in the input box and clicking the "Add" button or pressing Enter.
- Task Completion: Conveniently mark tasks as completed with a simple click.
- Task Removal: Effortlessly remove tasks by clicking the designated delete icon next to each task.
- Responsive Design: Enjoy a seamless experience across a variety of devices with our responsive design.

## 3. Getting Started

1. Repository Cloning:
   ```bash
   git clone https://github.com/karansingh21202/todo-list.git
   ```

2. Application Opening:
   Launch the `index.html` file in your preferred web browser.

## 4. Usage

- Task Addition:
   - Enter your task in the input box.
   - Click the "Add" button or press Enter to add it to the list.

- Task Management:
   - Mark tasks as completed with a click.
   - Remove tasks using the delete icon.

## 5. Project Structure

- `index.html`: The primary HTML file.
- `src.css`: The stylesheet responsible for application styling.
- `to.js`: The JavaScript file containing the application logic.
